The goal is to optimize service time, reduce errors, and improve overall business organization in an increasingly demanding digital market.

## üöÄ Key Features
The application includes several core modules for restaurant administration:
* **User Management:** Secure login and registration systems to control access.
* **Table Management:** Ability to create, edit, delete, and visualize the status (Available/Occupied) of the business's tables.
* **Product & Menu Management:** A digital menu organized by categories with details on ingredients, allergens, and pricing. Categories can be personalized with specific colors.
* **Order Management (Comandas):** Real-time recording of orders for each table with status tracking (Pending, Preparing, Ready, Delivered).
* **Invoice Generation:** An automated billing system that summarizes orders for a final price per table, allowing users to mark them as paid.
